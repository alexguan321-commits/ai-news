📄 值得关注的新论文（3 篇）

1. HANDOFF: 人形机器人全身控制的任务空间接口 (cs.RO / cs.AI) 核心贡献：提出了一种新的任务规划与全身控制之间的接口设计，让规划器不需要生成密集的运动学参考，而是通过高层语义指令控制人形机器人。 为什么重要：人形机器人落地的关键瓶颈之一就是"大脑"和"小脑"之间的接口设计。这个工作如果能在实际部署中验证，将大大加速人形机器人从实验室走向工厂和家庭的进程。

2. RREDCoT: 推理模型的段级奖励重分配 (cs.LG / cs.AI) 核心贡献：针对 GRPO 等 RL 微调方法，提出了在推理链的不同段落之间更精细地分配奖励信号，而不是只在最终答案上打分。 为什么重要：当前的推理模型训练（如 DeepSeek-R1、o1 系列）都在用类似 GRPO 的方法。段级奖励分配可以让模型更快学会"在哪些步骤上犯错"，提升推理训练效率——这直接影响下一代推理模型的质量。

3. Latent Reasoning with Normalizing Flows: 用连续隐空间做推理 (cs.CL / cs.LG) 核心贡献：提出用归一化流在连续隐空间中进行推理计算，而不是通过离散的 token 序列生成 Chain-of-Thought。 为什么重要：当前的 CoT 推理受限于 token 的离散性和串行性。如果能用连续隐空间做"内部思考"，推理速度和表达能力都可能有质的飞跃——这是通向"真正的推理"的一条新路径。
