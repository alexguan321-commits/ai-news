📄 学术前沿

1. EEVEE: 面向真实世界的测试时 Prompt 学习框架 来源：arXiv 2606.11182

首个支持多数据集测试时 prompt 学习的 LLM Agent 框架。核心突破：让 Agent 在真实任务流中持续自我改进，而非只在单一数据集上优化。这直接解决了 Agent 从 demo 到生产的 gap。

为什么重要：Agentic AI 的核心瓶颈之一是部署后如何持续优化。EEVEE 提供了系统性的解法。对 Alex 的 Dify/Agent 部署有直接参考价值。

2. TRACE: Agentic RL 的统一 Rollout 预算分配框架 来源：arXiv 2606.11119

解决 RLVR（可验证奖励的强化学习）中的核心效率问题：当 prompt 过于简单或复杂时，reward 对比度不足，rollout 计算资源被浪费。TRACE 动态分配预算，让训练效率大幅提升。

为什么重要：RL 是训练 reasoning agent 的主流路径。TRACE 让同样的算力能训练出更好的 agent。

3. Decentralized Multi-Agent Systems with Shared Context 来源：arXiv 2606.10662

提出去中心化多智能体系统架构，通过共享上下文替代中心编排。解决了 MAS 规模化时的单点瓶颈问题。

为什么重要：当前大多数 Agent 框架（包括 CrewAI、AutoGen）都是中心化的。这篇论文指向了更 scalable 的方向。

4. SkillAxe: 通过评估引导的自我精炼提升 LLM Agent 技能 来源：arXiv 2606.10546

发现 LLM 自己写的 skill 文档在 SkillsBench 上比人类写的差 16.2 个百分点。SkillAxe 通过评估反馈让 LLM 自我迭代 skill 质量，显著缩小差距。
