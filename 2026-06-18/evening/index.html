1. CodeSentinel：三层防御间接 Prompt 注入攻击

核心贡献：针对代码 LLM 从外部仓库检索上下文时的注入攻击面，提出三层防御架构。 为什么重要：Agent 从代码库/文档中检索上下文是标准操作，这篇论文直击 Agent 安全的核心漏洞。 🔗 https://arxiv.org/abs/2606.19235

2. PhantomSkill：Agent Skill 生态的恶意代码注入攻击

核心贡献：展示如何在 Skill 的辅助资源（而非文本指令）中隐藏恶意行为，绕过安全检测。 为什么重要：这是对 Agent Skill 生态的直接威胁。NVIDIA 同日推出 SkillSpector 不是巧合——行业已经在应对。 🔗 https://arxiv.org/abs/2606.19191

3. DreamReasoner-8B：Block Diffusion 推理模型

核心贡献：开源 8B 参数的 block diffusion 推理模型，通过 block-size curriculum learning 实现长链推理。 为什么重要：Diffusion 语言模型正在挑战 autoregressive 的统治地位，这篇论文证明 diffusion 可以做可靠的长链推理。 🔗 https://arxiv.org/abs/2606.19257

4. LLM Agent 通信协议技术分类学

核心贡献：系统梳理当前碎片化的 Agent 通信协议生态，提出互操作性框架。 为什么重要：多 Agent 系统的标准化是产业化的前提。这篇论文是"Agent 互联网"的 TCP/IP 时刻。 🔗 https://arxiv.org/abs/2606.19135
