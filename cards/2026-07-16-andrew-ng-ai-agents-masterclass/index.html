Andrew Ng AI Agents Masterclass: 2小时免费课程

Andrew Ng（吴恩达）发布了免费的 2 小时 masterclass，专门讲解如何使用 Anthropic 的 Claude 构建 AI agents。这标志着 AI agent 开发从理论研究进入实战部署阶段。

📚 课程核心内容

1. Build reusable agent skills with Claude

  • 创建可复用的 agent 技能模块
  • 技能封装和抽象

2. Use Anthropic's built-in skills

  • 利用 Anthropic 预置的技能系统
  • 开箱即用的 agent 能力

3. Understand skills vs MCP, tools & subagents

  • Skills：可复用的高级能力模块
  • MCP (Model Context Protocol)：标准化的工具调用协议
  • Tools：具体的工具实现
  • Subagents：子 agent 协调

4. Design long-running autonomous agents

  • 长时间运行的自主 agent 架构
  • 状态管理和容错机制

🎯 Skills vs MCP vs Tools vs Subagents

概念 定义 使用场景
Skills 可复用的高级能力模块 复杂任务的抽象和封装
MCP 标准化工具调用协议 跨模型/跨平台的工具互操作
Tools 具体的工具实现 文件操作、API 调用、计算等
Subagents 子 agent 协调 多 agent 协作、任务分解

💡 对 FDE 模式的启示

1. Agent Skills 是 FDE 的核心能力

  • FDE 需要快速构建可复用的 agent 技能
  • Anthropic 的 skills 系统提供了标准化框架

2. MCP 是工具互操作的标准

  • FDE 应该采用 MCP 协议确保工具的可移植性
  • 避免 vendor lock-in

3. 长时间运行 agent 是生产级应用的关键

  • 从 demo 到 production 需要容错和监控
  • FDE 需要掌握 agent 运维技能

🚀 创业机会

1. Agent Skills 市场 — 可复用的 skills 可以形成 marketplace 2. MCP 工具集成服务 — 帮助企业将现有系统封装为 MCP 工具 3. Agent 运维平台 — 监控、调试、优化长时间运行的 agents

🔗 知识关联

  • FDE 模式：Agent skills 是 FDE 快速交付业务价值的核心能力
  • Loop Engineering：长时间运行 agent 需要 loop 设计和状态管理
  • Context Engineering：Agent 的上下文管理是成功的关键
  • Ontology：Skills 的分类和组织需要知识图谱支持
