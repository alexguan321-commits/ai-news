Start a 1-Person Business with Claude (FULL COURSE)

Author: Ronin (@deronin_) Date: July 13, 2026 Engagement: 120 likes | 7 reposts | 11 replies | 33K+ views

Core Thesis

"I run a $40k MRR agency with zero employees. No developers. No sales team. No ops manager. Just Claude Code, a folder of skills organised like a real company, and a client pipeline that fills itself while I sleep."

Why 2026 Is the Cheat Code Year

  • 84% of the planet has never used AI once
  • 16% touched a free chatbot and stopped there
  • 0.3% pay $20/mo for a real model
  • A rounding error knows tools like Claude Code exist

Language arbitrage: Swedish, Portuguese, Greek, Polish inboxes are EMPTY while US inboxes are a warzone. Same offer, translated, multiples of the reply rate.

The Business Model (3 Levels)

Level 1: AI-Built Websites

  • Target: 70% of US small businesses with BAD websites (not the 30% with no website)
  • Bonus segments: brand-new businesses (fresh LLC registrations are public data), businesses running ads into a bad site
  • Typical ticket: ~$500 first sale, $3-10k once you have proof

Level 2: Automations and Agents

  • Invoicing, lead follow-up, onboarding, review collection, support
  • The cheat code: It's the same 4 automations in every niche: instant follow-up, review collection, reminders, dead-lead reactivation
  • Build them once, skill-ify them, deploy per niche forever
  • Typical ticket: $1-5k setup + $200-500/mo

Level 3: Full AI Systems

  • 5-25 automations bundled behind ONE dashboard
  • Client sees executions, hours saved, money saved
  • Typical ticket: $10-20k projects or $2-5k/mo retainers

The AI Operating System (20-Minute Setup)

Your entire company lives in ONE folder:

  • Claude Code = the operating system
  • Skills = the employees
  • The folder = the office

Core Principle:

"The IP of a one-person business is not code. It's your SOPs, your processes and your skills. Every time you build something twice, turn it into a skill. By month 6 your skills folder is worth more than your client list."

Client Machine #1: Personal Brand + Lead Magnets

Proof-First Principle

You don't need clients to build a track record. You need BUILDS.

1. Build Before Anyone Asks - Pick ONE niche, build 3-5 real demos for businesses in that niche (fake clients, real builds) 2. Post Every Build - Every build becomes 3-5 posts (screen recordings, before/after, numbers, how-I-did-it) 3. Lead Magnet - Template packs, audits, calculators. Make them COMMENT or DM to get it. 4. Demo-First Close - Find business with bad website → rebuild homepage → deploy on free subdomain → send Loom

Real Example

One tweet got 765 interested people. 300 read the docs. At 1% conversion × $247 = $741 from one tweet. Total: 683 members × $247 = $168,701.

Client Machine #2: Cold Email

The Stack

  • Sending tool with pre-warmed domains (~$10-15 per domain)
  • Each inbox sends ~20 emails/day safely
  • One domain = ~100 sends/day

Building the List


[niche] x [country] x [1-50 headcount] x [owner/founder titles] x [verified emails only]
  • Headcount 1-50 = owner reads their own inbox
  • Expect ~60-65% "good" emails, send ONLY to good

MCP Integration

Connect sending tool's MCP to Claude → Claude can create campaigns, write sequences, pull replies, manage follow-ups (~38 tools worth of control)

The Numbers

  • Good campaign: ~10,000 sends, 5%+ reply rate (industry norm ~0.5%)
  • 100 sends/day = one interested lead every other day
  • One interested lead every other day = full calendar within a month

The Sales System (Doctor Approach)

Before the Call

  • NEVER give price before the call
  • Confirmations + reminders on every channel
  • Never sell in DMs

During the Call

  • You talk 20%, they talk 80%
  • Question ladder: make them state the pain themselves
  • Show demo under 10 minutes
  • State price. ONE number, clean. Then SHUT UP.

The Claude Trick

Connect AI notetaker MCP (Fathom) to Claude. After the call: 


Read this transcript. Write the spec. Start building.

Zero context loss between what the client said and what gets built.

Delivery Framework

The Biggest Idea

"BUILD ONCE. SKILL-IFY IT. DELIVER FOREVER."

After any successful build: 1. Extract the process into a skill 2. Check if skill is standalone 3. Your skills folder = your product catalog = the actual IP

The 4-Step Framework to Build ANYTHING

1. BUILD PLAN - What you're building + tech stack + design/UX 2. IMPLEMENTATION PLAN - Build plan broken into ~40 steps with checkboxes 3. BUILD - Feed steps one at a time, approve between steps 4. TEST AND REFINE - Paste every error to Claude verbatim

The Ratio Rule: 80% of time goes into steps 1 and 2. 20% into step 3. Everyone does the opposite.

Two-Model Verification: One AI proposes the stack and plan. A second session audits it. Consultant + auditor catch each other's blind spots.

Pricing (The 5x Rule)

"GIVE EVERY CLIENT A 5X RETURN ON WHAT THEY PAY YOU. Save them $10k/mo, charge $2k/mo."

Three Pricing Structures

1. Result-based (~20% of gross profit) - Highest ceiling, requires trackability 2. Upfront + recurring (default) - $2k upfront + $500/mo service fee 3. Usage tiers - Like software: 100/1000/5000 calls

The Golden Rule of Messaging

  • BAD: "an AI agent with RAG over your knowledge base and confidence-scored escalation"
  • GOOD: "your support inbox answers itself and you save $10k a month"

Nobody buys AI. Nobody buys automation. They buy more money, more time, fewer employees doing boring work.

The Flywheel


build for a client → post the proof → proof pulls inbound leads → 
inbound = no connects, no sending costs → better clients → 
better builds → better proof → repeat

Action Steps

1. Set up the OS today: folder + Claude Code + skills + GitHub backup (20 minutes) 2. Build first template website tomorrow for a fake business 3. This weekend: create freelance profile AND launch one small cold campaign 4. Spend 4 HOURS A DAY on outreach - Marketing and selling are the real skills 5. Take first clients free or cheap - buying testimonials, trust, reps 6. Get to $2-3k/mo that covers your life 7. Then raise prices. Fire broke and annoying clients. 8. Find the bottleneck, fix it, find the next one.

Key Insight

"A business used to be an office full of people. Now it's a folder, a subscription, and someone who refuses to quit."
