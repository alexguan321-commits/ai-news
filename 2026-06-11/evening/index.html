3. Breaking Entropy Bounds: 用 MTP + Rejection Sampling 加速 RL 训练 来源：arXiv 2606.12370 RL rollout 是 LLM 训练瓶颈。证明 Multi-Token Prediction 结合 rejection sampling 可以突破 entropy 限制加速 rollout。训练效率提升方向。

4. OCELOT: LLM Agent 的推理泄漏预算 来源：arXiv 2606.12341 LLM Agent 代替用户操作时，每一步都可能泄漏 PII。提出跨步骤的隐私泄漏预算框架。对 Agent 安全部署有直接价值。

🦾 具身智能 & 机器人

今天 arXiv 具身方向论文密集，三个值得关注的方向：

  • FACTR 2（2606.12406）：用数据驱动方法从普通机械臂估算外力矩，无需专用力传感器。降低了力控操作的硬件门槛。
  • World Pilot（2606.12403）：用世界-动作先验引导 VLA 模型，解决 VLA 在接触密集操作中的动态理解不足。
  • Ambient Diffusion Policy（2606.12365）：从次优数据中学习模仿——高质量机器人数据太贵，这个方法允许用低质量/分布外演示训练。

趋势： VLA（Vision-Language-Action）模型仍是主战场，但关注点正从"能不能做"转向"怎么做得更高效、更鲁棒、更低成本"。
