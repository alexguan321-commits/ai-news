📄 学术前沿

1. Recursive Agent Harnesses（上面已述） 核心贡献：命名并形式化了递归 Agent 模式，分析了 Anthropic dynamic workflows 与学术 RLM 之间的共同结构。为什么重要：这是 Agentic Engineering 的基础理论建设。

2. Operadic Consistency: Label-free Signal for Reasoning Failures in LLMs 来源：arXiv 2606.13649 核心贡献：用 Operad 理论（组合系统的形式化框架）检测 LLM 推理失败，无需 ground-truth labels。为什么重要：为 LLM 推理质量提供了一种数学上优雅的 self-check 机制，可能替代 self-consistency / semantic entropy 等现有方法。

3. Valid Inference with Synthetic Data via Task Exchangeability 来源：arXiv 2606.13629 核心贡献：为合成数据（包括 LLM 生成的"硅样本"）提供统计推断有效性框架。为什么重要：当企业开始用 LLM 生成模拟数据做决策时，这篇论文回答了"这些结论在统计上是否站得住脚"。

4. Agents-K1: Agent-native Knowledge Orchestration 来源：arXiv 2606.13669 核心贡献：将科学论文从扁平引用图升级为包含 entities, claims, evidence, mechanisms 的知识编排系统。为什么重要：直接关联 Ontology-driven AI，是知识工程在 Agent 时代的落地方案。
