1. The Value Axis: Language Models Encode Whether They're on the Right Track

核心贡献：发现 Qwen3-8B 内部存在一个"价值轴"，能追踪当前策略达成目标的概率。这意味着模型在推理时内部有"方向感"。

🔗 arXiv:2606.17056

2. TokenPilot: Cache-Efficient Context Management for LLM Agents

核心贡献：提出一种保持 KV cache 命中率的上下文管理方法，解决长会话中 token 成本飙升的问题。

为什么重要：这是 Agentic Engineering 的基础设施问题——Agent 运行越久成本越高。TokenPilot 让长时运行的 Agent 更经济。

🔗 arXiv:2606.17016

3. ContextRL: Context-Aware RL for Agentic and Multimodal LLMs

核心贡献：提出 ContextRL，让 LLM 在长上下文或多模态输入中定位关键证据的能力通过 RL 强化。

为什么重要：解决了 Agent 的"大海捞针"问题——在复杂工具输出或长文档中找到决定性证据。

🔗 arXiv:2606.17053

4. T-Rex: Tactile-Reactive Dexterous Manipulation

核心贡献：提出一种让 VLA 模型动态响应触觉信号的灵巧操作框架，突破现有模型只能处理静态触觉编码的限制。
