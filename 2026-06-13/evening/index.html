📄 学术前沿

1. Learning to Reason by Analogy via Retrieval-Augmented Reinforcement Fine-Tuning 来源：arXiv 核心贡献：传统 RAG 基于语义相似度检索，但语义相似的问题可能需要完全不同的推理路径。本文用 reinforcement fine-tuning 训练 retriever 学习"类比检索"——检索结构相似而非表面相似的问题。 为什么重要：这是 RAG 从"知识检索"升级到"推理检索"的关键一步。对 enterprise RAG（如某跨国消费品巨头供应链问题检索）有直接启示。

2. Operadic Consistency：无标签检测 LLM 组合推理失败 来源：arXiv 核心贡献：用 Operad 理论（组合系统的形式化框架）检测 LLM 推理中的组合性失败，无需 ground-truth labels。 为什么重要：inference-time 的 reasoning quality 检测一直是难题。这个方法提供了一个数学上优雅的 confidence signal 替代方案。

3. Can I Buy Your KV Cache？ 来源：arXiv 核心贡献：指出当前 AI agent 的荒谬现状——每个 agent 都在重复 prefill 相同文档，重建相同 KV cache。提出 KV cache 共享/交易市场。 为什么重要：如果 agent 规模上去了，KV cache 重复计算是巨大的成本浪费。这篇论文提出了一个基础设施层面的优化方向。
