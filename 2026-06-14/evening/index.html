📄 学术前沿

1. HyperTool: Beyond Step-Wise Tool Calls for Tool-Augmented Agents 核心贡献：指出当前 Agent 工具调用的"执行粒度失配"问题——局部确定性的工具流程被展开成重复的模型可见步骤。提出 HyperTool 抽象，将多步工具调用封装为单一语义操作。 为什么重要：这直接影响你在 Dify 中搭建的 Agent 工作流效率。如果工具调用能从 step-wise 升级为 composite，推理成本和延迟都会显著下降。 🔗 https://arxiv.org/pdf/2606.13663v1

2. Recursive Agent Harnesses 核心贡献：命名并系统化研究"递归 Agent 框架"模式——Agent 编写代码来生成子 Agent，形成递归调用链。与 Anthropic 的 dynamic workflows 实践呼应。 🔗 https://arxiv.org/pdf/2606.13643v1

3. Operadic Consistency: 无标签信号检测 LLM 组合推理失败 核心贡献：用 Operad 理论（迭代组合系统的形式化）构建无标签的推理失败检测信号，替代 self-consistency、semantic entropy 等方法。 为什么重要：推理可靠性是 Agent 落地的核心瓶颈。如果能在推理时自动检测组合推理失败，就能构建更可靠的 Agent 容错机制。 🔗 https://arxiv.org/pdf/2606.13649v1
